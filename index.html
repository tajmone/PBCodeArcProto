PBCodeArcProto

Pre-alpha drafting stage

Temporary prototype dev repo for a website generator for the PureBasic CodeArchiv Rebirth project. Will be destroyed once the Indexer is ready and integrated into the parent project.

Based on the original discussion with @SicroAtGit in Issue #5.

Project organization:

  • /_assets/ — the actual HTMLPagesCreator code + HTML assets
  • /_tempwork/ — isolated code work

the remaining folders are contents taken from the PureBasic CodeArchiv Rebirth project to test the app.
